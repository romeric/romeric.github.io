High Order Curvilinear Mesh Generation:

A core part of my research work focuses on implementing novel algorithms for generating structured and unstructured meshes for high order finite element analysis. In particular, I work on various solid mechanics based analogies for a posteriori mesh generation.

Further details about this project can be found in the PostMesh official GitHub repository
