How fast can you finite element?

Writing a high performance finite element assembler for nonlinear coupled problems, such as those emanating from a mixed variational principle for large strain electro-magneto-mechanics is a challenging task and beyond a point becomes forbiddingly complex for automated code generators. In this blog, I will talk about some novel compile time meta-programming techniques for explicit manipulation of finite element matrices/tensors using modern C++ features, which can perform operation minimisation using graph optimisation algorithms and further convert scalar codes to their data parallel (SIMD) variants to achieve the theoretical peak FLOP performance.
